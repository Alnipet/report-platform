# Report Platform - Frontend

Клиентское приложение для платформы генерации отчётов. SPA написано на React и собрано с помощью Vite.

## Описание
Этот модуль отвечает исключительно за пользовательский интерфейс:
- Отображение списка доступных отчётов, получаемых с бэкенда.
- Отправка запросов на асинхронную генерацию отчёта.
- Опрос (Polling) статуса готовности отчёта.
- Предоставление возможности скачивания готового файла.

## Стек технологий
- **React 19**
- **Vite**
- **Tailwind CSS** + компоненты **shadcn/ui**
- **TypeScript**

## Запуск вне Docker
Если вы хотите запустить фронтенд локально (без `docker-compose`):

```bash
npm install
npm run dev
```

> **Примечание**: Фронтенд ожидает, что API работает на порту 3001. Если API запущено по другому адресу, переопределите переменную среды `VITE_API_URL`.
